# Modern Online Banking Authentication System (Tkinter, AI MFA)

A desktop banking authentication demo with **multi-factor authentication**:
- Registration with email/password/mobile/passphrase
- Face enrollment & verification (OpenCV LBPH)
- Email OTP (via Gmail SMTP)
- Voice verification (SpeechRecognition with fallback)
- Async subprocesses keep Tkinter UI responsive

See Modern_Banking_Auth_System_Guide.txt for detailed explanation and interview-style Q&A.
